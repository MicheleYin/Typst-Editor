# App-bundled fonts

Place `.ttf`, `.otf`, `.ttc`, or `.otc` files here. They are shipped with the app (via Tauri `bundle.resources`) and loaded by the Typst compiler **in addition to** the default fonts from `typst-assets` (e.g. New Computer Modern).

Subfolders are scanned recursively.

**Licensing:** Only add fonts you are allowed to redistribute.
